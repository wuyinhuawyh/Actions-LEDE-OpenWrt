### 简介
编译一些设备的OpenWrt固件，**仅满足自己学习**

默认LAN IP:192.168.2.1    
默认用户名root，密码password

### 设备
x86_64、Redmi_AC2100、竞斗云、Newifi3-d2、WR1200JS、 HIWIFI-C625A
